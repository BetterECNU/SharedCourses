### 三、简答题（每题 5 分，共 10 分）

1. 请简述 Java 程序从编写到编译，再到运行的全过程，列出中间的使用的命令。

    <details>
    <summary>答案：</summary>

    使用文本编辑器（如 Notepad++、Text 等）编写 Java 源代码，保存为 `.java` 扩展名的文件。使用 Java 编译器 `javac` 将源代码编译成字节码文件（`.class` 文件）。Java 命令 `java` 运行编译后的字节码文件。注意包 `package` 的使用。

    </details>

    ***

2. 请说明 `final` 关键字的作用。

    <details>
    <summary>答案：</summary>

    `final` 关键字是一个修饰符，用于指定一个变量、方法或类不能被进一步修改或扩展。当应用于局部变量或实例变量时，`final` 表示该变量一旦被初始化后，其引用或值就不能被改变。当一个方法被声明为 `final`，则该方法不能在任何子类中被重写（override）。当一个类被声明为 `final`，则该类不能被继承。

    </details>

5. 请阅读以下程序。如果程序有错误，请列出错误，并纠正给出运行结果。如果没有，直接写出结果。

    ```java
    (01) class Parent {
    (02)     String a;
    (03)     public void Parent(String a) {
    (04)         this.a = a;
    (05)     }
    (06)     public void print() {
    (07)         System.out.println(a);
    (08)     }
    (09) }
    (10)
    (11) public class Child extends Parent {
    (12)     String b;
    (13)     public Child(String a, String b) {
    (14)         this.b = b;
    (15)         super(a);
    (16)     }
    (17)     public void print() {
    (18)         System.out.println(a + " to " + b);
    (19)     }
    (20)     public static void main(String arg[]) {
    (21)         Parent p = new Child("east", "west");
    (22)         p.print();
    (23)     }
    (24) }
    ```

    <details>
    <summary>答案：</summary>


    </details>
