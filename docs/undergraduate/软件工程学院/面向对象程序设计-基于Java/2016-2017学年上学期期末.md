4. JAVA 语言支持数组操作，以下语句哪一句是不正确的？

    </details>

    ```java
    public class SimpleClass{
        double d;
        public SimpleClass(double t){
            d=t;
        }
        public void method(){
            System.out.println("the result is"+d);
    ```

3. 该程序有两个错误，请指出并改正。

    ```java
    package package2;
    public class person{
        String name;
        public person(){
            name="";
        }
        public person(String n){
            name=n;
        }
        protected void setName(String n){
            name=n;
        }
    }

    package package1;
    import package2.*;
    public class student extends person{
        person p;
        public student(){}
        public void setName(String n){
            super.setName(n);
            p=new person();
            p.setName();
        }
    }
    ```

    <details>
    <summary>答案：</summary>

    `protected void setName()` 只能在相同包内访问，应为 `public`。

    `p.setName();` 缺少参数。

    </details>
