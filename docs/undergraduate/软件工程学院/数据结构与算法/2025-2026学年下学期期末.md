### 二、简答题（本大题共 4 小题，每题十分）

**1. 循环时间复杂度推导**

算法循环控制变量 `i` 初始值为 `0`；
迭代增长规则：
- 第1步：`i += 1`
- 第2步：`i += 2`
- 第3步：`i += 3`
依次累加，终止条件 `i > n` 时退出循环。

要求：详细写出推导过程，给出大O时间复杂度。

<details>
<summary>解：</summary>

第 $k$ 步后，$i = 1 + 2 + \cdots + k = \frac{k(k+1)}{2}$。
终止条件为 $\frac{k(k+1)}{2} > n$，即 $k^2 + k - 2n > 0$。
解得 $k \approx \sqrt{2n}$，故时间复杂度为 $O(\sqrt{n})$。

</details>

**2. 快速排序(Quicksort)过程演练**

原始无序序列：`[28, 15, 42, 9, 67, 33, 21, 54]`
要求：
1. 基准值选取规则：`floor((low + high) / 2)` 向下取整对应元素
2. 写出每一趟 Partition 划分后的完整序列或者绘制完整递归调用树展示分治流程

<details>
<summary>解：</summary>

序列下标 0~7，初始 low=0, high=7，基准下标 = floor((0+7)/2) = 3，基准值 = 9。

第一趟（基准 9）：划分后 `[9, 15, 42, 28, 67, 33, 21, 54]`，基准位置 0。
左半为空，右半 low=1, high=7，基准下标 = floor((1+7)/2) = 4，基准值 = 67。

第二趟（基准 67）：划分后 `[9, 15, 42, 28, 54, 33, 21, 67]`，基准位置 7。
左半 low=1, high=6，基准下标 = floor((1+6)/2) = 3，基准值 = 28。

第三趟（基准 28）：划分后 `[9, 15, 21, 28, 54, 33, 42, 67]`，基准位置 3。
左半 low=1, high=2，右半 low=4, high=6。

左半（基准 15）：划分后不变 `[9, 15, 21, ...]`，基准位置 1。

右半（基准 33）：划分后 `[..., 28, 33, 54, 42, 67]`，基准位置 5。

继续递归直至有序。最终结果：`[9, 15, 21, 28, 33, 42, 54, 67]`。

</details>

**4. Dijkstra 最短路径算法分析**

有向带权图邻接矩阵（∞代表无直接边）：

| 起点 \ 终点 | A | B | C | E |
| ---- | ---- | ---- | ---- | ---- |
| A | 0 | 10 | ∞ | 100 |
| B | ∞ | 0 | 50 | ∞ |
| C | ∞ | ∞ | 0 | 10 |
| D | ∞ | ∞ | 20 | 60 |
| E | ∞ | ∞ | ∞ | 0 |

源点指定为顶点A，要求：
1. 完整求解步骤，绘制状态更新表格（记录集合S、各顶点距离动态更新）
2. 输出A到其余所有顶点的最短路径与对应距离

<details>
<summary>解：</summary>

| 步骤 | S | dist(A) | dist(B) | dist(C) | dist(D) | dist(E) |
|------|---|---------|---------|---------|---------|---------|
| 初始化 | \{A\} | 0 | 10 | ∞ | ∞ | 100 |
| 1. 选 B | \{A,B\} | 0 | 10 | 60(via B) | ∞ | 100 |
| 2. 选 C | \{A,B,C\} | 0 | 10 | 60 | ∞ | 70(via C) |
| 3. 选 E | \{A,B,C,E\} | 0 | 10 | 60 | ∞ | 70 |
| 4. 选 D | \{A,B,C,E,D\} | 0 | 10 | 60 | ∞ | 70 |

注：顶点 D 不可达（没有边指向 D），dist(D) = ∞。

最短路径：
- A→B：距离 10
- A→C：路径 A→B→C，距离 60
- A→D：不可达，距离 ∞
- A→E：路径 A→B→C→E，距离 70

</details>

### 三、代码题（本大题共 2 小题，每题十分）

**1. 单链表相邻节点成对指针翻转**

功能：交换第 `2i-1` 和 `2i` 个相邻节点，**仅修改指针，禁止交换节点数值**，返回新头指针。
输入示例：`1 → 2 → 3 → 4 → 5`
输出示例：`2 → 1 → 4 → 3 → 5`

节点结构体：
```c
struct ListNode {
    int val;
    struct ListNode *next;
};
```
函数框架：
```c
struct ListNode* swapPairs(struct ListNode* head) {
    // 编写实现代码
}
```

<details>
<summary>解：</summary>

```c
struct ListNode* swapPairs(struct ListNode* head) {
    struct ListNode dummy = {0, head};
    struct ListNode *prev = &dummy;
    while (prev->next && prev->next->next) {
        struct ListNode *first = prev->next;
        struct ListNode *second = first->next;
        first->next = second->next;
        second->next = first;
        prev->next = second;
        prev = first;
    }
    return dummy.next;
}
```

</details>

**2. 二叉树任意节点间最大路径长度**

功能：求二叉树内部最长路径跨度，路径不一定经过根节点。
示例说明：
树结构：
```
        1
       / \
      2   3
     / \   \
    4   5   6
```
最长路径：`4 → 2 → 1 → 3 → 6`，路径长度为4。

节点结构体：
```c
struct TreeNode {
    int val;
    struct TreeNode *left;
    struct TreeNode *right;
};
```
函数框架：
```c
int maxPathSum(struct TreeNode* root) {
    // 编写实现代码
}
```

<details>
<summary>解：</summary>

```c
int dfs(struct TreeNode* root, int *ans) {
    if (!root) return 0;
    int left = dfs(root->left, ans);
    int right = dfs(root->right, ans);
    if (left + right > *ans) *ans = left + right;
    return (left > right ? left : right) + 1;
}

int maxPathSum(struct TreeNode* root) {
    int ans = 0;
    dfs(root, &ans);
    return ans;
}
```

</details>
